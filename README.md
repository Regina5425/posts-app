<!-- GETTING STARTED -->
### Установка

1. Склонировать репозиторий
   ```sh
   git clone https://github.com/Regina5425/posts-app
   ```
2. Установить NPM пакеты
   ```sh
   npm install
   ```
3. Запустить приложение на localhost
   ```js
   npm start
   ```
